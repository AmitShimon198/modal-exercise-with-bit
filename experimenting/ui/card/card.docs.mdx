## React Component for rendering text

A basic div that renders some text

### Component usage
```js
  <Card >
    <Header className={classes.fancyCardHeader}>
      Example Card
    </Header>
    <Hr/>
    <MainContent className={classes.fancyCardMainContent}>
      Main Content
    </MainContent>
    <Button onClick={()=>alert('OK')}><span>Ok</span></Button>
    <Button onClick={()=>alert('Cancel')}><span>Cancel</span></Button>
  </Card>
```

### Using props to customize the text

Modify the text to see it change live:
