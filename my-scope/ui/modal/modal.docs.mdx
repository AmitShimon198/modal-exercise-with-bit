import { Modal } from './modal';

## React Component for rendering text

A basic div that renders some text

### Component usage
```js
<Modal modalTitle={'Example modal'} portalElement={portalElement} onClose={toggleExternally} isOpen={openExternally} >
      <h1>Hi From modal</h1>
      <Button onClick={onConfirm}><span>Confirm</span></Button>
</Modal>
```

### Using props to customize the text

Modify the text to see it change live:
